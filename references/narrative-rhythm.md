# 叙事节奏系统

> 让 PPT 从"独立页面的堆砌"变成"一气呵成的演讲"。

## 核心概念：页面视觉重量

每种页面类型有固定的"视觉重量"分值，代表观众需要消耗的注意力：

| 页面类型 | 重量分 | 说明 |
|---------|-------|------|
| 封面页 | 8 | 高冲击力但信息量不大 |
| 目录页 | 4 | 导航性质，快速扫过 |
| 章节封面 | 2 | 纯呼吸页，最低密度 |
| 数据密集页（混合网格） | 9 | 最高认知负荷 |
| 标准内容页（主次/英雄式） | 7 | 中高认知负荷 |
| 对比页（50/50） | 6 | 中等，结构化对比降低负荷 |
| 单一焦点页 | 5 | 单个核心信息 |
| 结束页 | 6 | 总结回顾 |

## 节奏规则

### 规则 1：相邻页重量差限制

相邻两页的重量差 **不超过 5 分**。避免"从极简章节封面(2)直接跳到数据密集页(9)"的断裂感。

**正确**：章节封面(2) -> 标准内容页(7) -> 数据密集页(9)
**错误**：章节封面(2) -> 数据密集页(9)（差 7 分，观众认知断裂）

### 规则 2：高密度页后必须降压

连续 **不超过 2 页** 高密度页（重量 >= 7）。第 3 页必须是中低密度（对比页/单一焦点页/章节封面），让观众"喘口气"。

### 规则 3：每个 Part 内的理想节奏

每个 Part（一个章节）内部遵循"开-展-深-收"四拍节奏：

```
    开场(2)     展开(7)      深入(9)       深入(7)     收束(5~6)
  章节封面  ->  总览页    ->  数据页    ->  内容页   ->  小结/亮点
  (呼吸)     (英雄式)    (混合网格)    (主次结合)    (单一焦点)
```

**简化版（Part 只有 2-3 页时）**：
```
  章节封面(2) -> 核心页(7~9) -> 结论页(5~6)
```

### 规则 4：全局起伏曲线

理想的整体 PPT 应形成"波浪曲线"，而非平直线或锯齿线：

```
         高
   9  |    ----         ----         ----
   7  |  /      \     /      \     /      \
   5  |/          \ /          \ /          \---
   2  |            v            v
         Part 1       Part 2       Part 3    End
```

## 章节色彩递进规则

每个 Part 使用不同的主 accent 色，让观众无意识感知章节切换：

| Part 序号 | 主 accent 色 | 用在哪里 |
|-----------|-------------|---------|
| Part 1 | `--accent-1` | 卡片竖线、标题高亮、数据数字、进度条 |
| Part 2 | `--accent-2` | 同上 |
| Part 3 | `--accent-3` | 同上 |
| Part 4 | `--accent-4` | 同上 |
| Part 5+ | 回到 `--accent-1`（循环） | 同上 |

**注意**：accent 色只影响页面内的"点缀"元素（竖线、数字、标签），不影响整体背景和卡片背景色。这样既有章节辨识度，又不破坏全局风格统一。

## 封面-结尾呼应清单

| 呼应维度 | 封面页 | 结束页 |
|---------|-------|-------|
| 角标装饰 | 左上+右下 | 右上+左下（对称） |
| 配图位置 | 右侧渐隐 | 右侧渐隐（相同技法） |
| 光晕位置 | 右上+左下 | 左上+右下（对称） |
| 标题下划线 | 有 | 有（相同宽度） |
| 品牌区域 | 底部左侧 | 底部左侧（相同位置） |
| 数据亮点 | 封面数据条 | 结束页不重复数据（改为要点回顾） |
