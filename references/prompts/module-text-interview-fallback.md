# Text Fallback Mode -- 结构化文本采访单

## 核心指令

当前 CLI 不支持原生结构化采访 UI，必须回退为**结构化文本采访单**。

禁止退化为：

- 一整行 `场景=；受众=；目标动作=...`
- 没有分组的散乱追问
- 只问 3-4 个字段的缩水版本

## 固定输出格式

```markdown
请按下面结构回复；不确定可以写“默认”。

**A. 场景与目标**
- 场景：
- 受众：
- 目标动作：

**B. 内容与边界**
- 4页密度：少而精 / 适中 / 信息量大
- 必含内容：
- 必避内容：
- 资料策略与分支：research / 非 research

**C. 风格与素材**
- 风格：
- 品牌限制：
- 语言：
- 配图：decorate / generate / provided / manual_slot
- 封面署名信息：要 / 不要；若要，请写内容

**D. 成功标准与执行**
- 成功标准：
- 子代理模型：跟主代理 / 指定模型
```

若需要极简回法，可在结尾补一句：

```text
如果你想省事，也可以直接回复：全部按默认，用 research。
```
