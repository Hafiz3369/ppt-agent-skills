# 封面页

> 封面是第一印象。以下是**建议结构**，你可以直接按此生成（快速模式），也可以自由发挥（只要确保封面有冲击力和核心信息传达）。

## 建议结构要素

| 要素 | 建议规格 | 灵活度 |
|------|---------|--------|
| 超大标题 | 48-56px, 粗体 | **必选** -- 封面没有大标题就不是封面 |
| 副标题/导语 | 18-20px | 建议有，可省略如果标题已足够清晰 |
| 标题装饰 | 下划线/色块/光标记等 | 建议有，形式自由 |
| 数据亮点 | 1-4 个大数字 | 建议有（用数据制造冲击力），纯叙事型可省略 |
| 品牌区域 | 演讲人/公司/日期 | 建议有，位置自由（底部/侧面/角落） |
| 配图 | 渐隐融合/色调蒙版/氛围底图 | 可选 |
| 装饰 | 从风格工具箱选 2-3 种 | 建议有，手法自由 |

## 构图参考（不限于此）

| 构图 | 特点 |
|------|------|
| 左文右图 | 最经典，信息清晰 |
| 居中对称 | 庄重感，适合正式场合 |
| 上图下文 | 图片主导，氛围强 |
| 全图叠文字 | 电影感，视觉冲击最强 |
| 纯文字 | 极简，适合学术/数据驱动型 |
| **自由构图** | 以上都不满意？随意发挥，只要标题醒目 |

## 唯一硬约束

- 所有颜色通过 CSS 变量引用
- 装饰用真实 DOM 节点（管线安全底线）
