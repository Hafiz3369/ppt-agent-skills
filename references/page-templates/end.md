# 结束页

> 结束页与封面呼应，总结核心要点。

## 建议结构

| 要素 | 建议规格 | 灵活度 |
|------|---------|--------|
| 总结标题 | 40-48px | **必选** |
| 要点回顾 | 3-5 条核心结论 | 建议有（无则页面太空） |
| 行动号召/联系方式 | CTA 或联系方式 | 可选 |
| 品牌区域 | 与封面位置呼应 | 建议有 |

## 设计原则

- 与封面形成视觉呼应（色调一致、构图对称/镜像）
- 不重复封面的数据，改为要点总结
- 有"收束感"而非"重新开始感"

## 唯一硬约束

- 所有颜色通过 CSS 变量引用
- 装饰用真实 DOM 节点
