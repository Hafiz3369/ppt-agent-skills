# 章节封面页

> 章节封面是「呼吸页」，视觉重量最低。作用是告诉观众"进入新章节"，同时让大脑放松。

## 建议结构

| 要素 | 建议规格 | 灵活度 |
|------|---------|--------|
| 章节编号 | "PART 01"/"第一章"/自定义 | **必选** |
| 章节标题 | 40-48px, 粗体 | **必选** |
| 导语 | 1-2 句预告 | 建议有，可省略 |

## 设计原则

- **大量留白** -- 内容占页面不超过 40-50%
- 不需要卡片和数据可视化
- 配图可选，用极低 opacity 做背景氛围
- 相邻章节封面的构图应有变化

## 唯一硬约束

- 所有颜色通过 CSS 变量引用
- 装饰用真实 DOM 节点
