# 目录页

> 目录页是导航。以下是**建议结构**，可直接使用或自由变化。

## 建议结构

| 要素 | 建议规格 | 灵活度 |
|------|---------|--------|
| 页面标题 | "目录"/"CONTENTS"/自定义 | **必选** |
| Part 列表 | 每个 Part 编号 + 标题 | **必选** -- 至少让观众知道有哪几个部分 |
| 页码/进度 | 页脚区域 | 建议有 |

## 布局参考

- 2-3 个 Part -> 横向大卡片或编号列表
- 3-4 个 Part -> 网格卡片
- 5+ 个 Part -> 紧凑编号列表
- **自由布局** -> 任何能清晰展示结构的方式

## 唯一硬约束

- 所有颜色通过 CSS 变量引用
- 装饰用真实 DOM 节点
